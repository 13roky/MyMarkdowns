## 什么是 JavaScrip

---

JavaScript 是一种脚本语言，这门语言可以用于 HTML 和 web，还可以广泛的适用于服务器、PC、笔记本电脑、平板电脑、和只能手机等设备。

## JavaScript 命名规范

---

### 标识符命令规范

- 变量、函数的命名必须要有意义
- 变量的名称一般用名词
- 函数的名称一般用动词

### 操作符规范

- 操作符左右两侧个保留一个空格

### 单行注释规范

- 一般在注释后面打一个空格然后在写注释

### 其它规范

- 小括号两边一般加空格
